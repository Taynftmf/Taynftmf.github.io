order graph

Full Usage: order graph

Parameters:
    graph : Graph<'a, 'b, 'c>

Returns: seq<Context<'a, 'b, 'c>>

Partially order vertices in a graph such that leaves appear first, then vertices pointing to leaves, then vertices pointing to these vertices and so on. Returns the full contexts. If viewing the given directed graph as modelling dependencies, this gives such vertices first which don't depend on any other vertices, making possible actions in a meaningful order. If the graph contains cycles, all vertices participating in a cycle are not part of the result. Check the number of returned contexts against the original number if that is important for you.

graph : Graph<'a, 'b, 'c>
Returns: seq<Context<'a, 'b, 'c>>
